## Installing PyRosetta
### After downloading, unzip PyRosetta's and enter the setup directory to install it
```
tar -xvzf PyRosetta[release].tar.bz2
cd PyRosetta*/setup
python3 setup.py install
```
## Installing aditional Python libs
```
pip install pandas
pip install numpy
```
